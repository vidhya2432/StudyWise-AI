# StudyWise AI

Your smart, AI-powered study companion built with Next.js, Genkit, and Tailwind CSS.

## Features
- **AI Notes Summarizer**: Upload PDF/PPT/Word files for summaries, flashcards, and quizzes.
- **Audio Study Mode**: Listen to your summaries with high-quality AI speech.
- **AI Study Planner**: Sync your timetable and organize subjects into dedicated workspaces.
- **AI Doubt Solver**: Get instant, clear explanations for difficult academic concepts.
- **Pomodoro Timer**: Stay focused with integrated productivity sessions.
- **XP & Gamification**: Earn XP and track your daily study goals.

## Troubleshooting Git Errors

If you see the error `fatal: 'origin' does not appear to be a git repository`, follow these steps:

1.  **Initialize Git** (if you haven't): `git init`
2.  **Link your Remote**: `git remote add origin <YOUR_REPOSITORY_URL>`
3.  **Set Branch to Main**: `git branch -M main`
4.  **Push your Code**: `git push -u origin main`

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **AI**: Google Genkit + Gemini 2.5 Flash
- **Styling**: Tailwind CSS + Shadcn UI
- **Storage**: Local Storage (Persistence without a backend)
