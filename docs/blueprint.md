# **App Name**: StudyWise AI

## Core Features:

- AI Study Planner: Generates a personalized daily study schedule, revision plan, and focus areas based on subjects, exam date, and daily free time using AI. 
- AI Doubt Solver: Provides simple explanations, examples, and practice questions for difficult concepts using AI.
- Smart Weakness Detection: Tracks quiz results, time spent per subject, and mistake patterns to identify weak areas and suggests specific revisions using AI tool.
- AI Notes Summarizer: Summarizes uploaded PDFs and text notes, generates flashcards, and creates quiz questions using AI.
- Productivity Tracker: Tracks daily streaks and study hours, displaying them on a graph with motivational messages from AI.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) for a sophisticated and intellectual feel.
- Background color: Light gray (#ECEFF1) to ensure readability and a clean interface in a light color scheme.
- Accent color: Soft teal (#4DB6AC) for highlighting key elements and CTAs.
- Headline font: 'Space Grotesk', a proportional sans-serif to bring a contemporary, techy feel to the app's headers.
- Body font: 'Inter', a grotesque-style sans-serif to keep a modern and neutral style that ensures optimal readability.
- Use minimalist icons to represent study topics, tools, and productivity metrics.
- Subtle transitions and animations to provide feedback during study planning and content generation.
